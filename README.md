## 📚 Sobre o Projeto

O **CalcMaster** é uma aplicação web interativa desenvolvida com **HTML, CSS e JavaScript**, criada como parte da disciplina de **Matemática Aplicada à Computação**.

O sistema tem como objetivo auxiliar estudantes na realização de cálculos acadêmicos e matemáticos de forma simples, intuitiva e visual, promovendo melhor compreensão dos conteúdos.

---

## ⚙️ Funcionalidades

- 📊 Cálculo de média escolar  
- 🎯 Cálculo da nota necessária para aprovação  
- ➗ Resolução de equações do 1º grau  
- 📈 Resolução de equações do 2º grau  

---

## 💻 Como usar o sistema

O **CalcMaster** foi desenvolvido para ser simples e intuitivo. Siga os passos abaixo para utilizar:

### 📊 Cálculo de média
1. Insira o nome do aluno  
2. Digite as três notas  
3. Clique em **"Calcular Média"**  
4. O sistema informará a média e a situação (Aprovado/Reprovado)

---

### 🎯 Nota necessária
1. Insira o nome e duas notas  
2. Clique em **"Nota para passar"**  
3. O sistema mostrará quanto falta para atingir a média

---

### ➗ Equação do 1º grau
1. Insira os valores de **a** e **b**  
2. Clique em **"Resolver"**  
3. O sistema exibirá o valor de **x**

---

### 📈 Equação do 2º grau
1. Insira os valores de **a**, **b** e **c**  
2. Clique em **"Resolver"**  
3. O sistema calculará as raízes da equação
